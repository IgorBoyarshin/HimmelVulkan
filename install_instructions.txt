==== WINDOWS ====
1) Download GLM into a folder; point .env.cmake into it
2) Download GLFW (windows pre-compiled binary) into a folder; point .env.cmake into it
3) Download LLVM (official website -> github -> LLWM-15-win64.exe -> Install it); Add to PATH "..\LLVM\bin"
4) Download Mingw (official website -> w64devkit -> github releases -> w64devkit-1.19.0.zip); Add to PATH "..\w64devkit\bin"
5) Download CMake, choose "add to Path for current user" during installation.
6) Get Models from somewhere and place into "models"
7) Create the "../shaders/out" folder manually so that Windows can create files there
